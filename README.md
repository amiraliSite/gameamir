# 🎮 NeoArena - فروشگاه تجهیزات گیمینگ

یک فروشگاه آنلاین مدرن برای تجهیزات گیمینگ با React و TypeScript.

## ✨ ویژگی‌ها

- 🌐 پشتیبانی از دو زبان (فارسی/انگلیسی)
- 🌙 حالت تاریک/روشن
- 🛒 سبد خرید کامل
- 📱 طراحی ریسپانسیو
- 🎨 UI مدرن با Tailwind CSS

## 🛠️ تکنولوژی‌ها

- React 18
- TypeScript
- Tailwind CSS
- Lucide Icons
- React Icons

## 🚀 نصب و اجرا

```bash
# کلون کردن پروژه
git clone https://github.com/YOUR-USERNAME/neoarena.git

# ورود به پوشه
cd neoarena

# نصب وابستگی‌ها
npm install

# اجرای پروژه
npm run dev
